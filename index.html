🔗 Google 試算表連結

連結後，資料會同步儲存到 Google 試算表，並可從任何裝置讀取。

建立 Google 試算表

前往 sheets.google.com，建立新試算表，命名為「撲克錦標賽」

開啟 Apps Script

在試算表中點選「擴充功能」→「Apps Script」

貼上以下程式碼

function doPost(e) { var ss = SpreadsheetApp.getActiveSpreadsheet(); var data = JSON.parse(e.postData.contents); var sheet = ss.getSheetByName(data.sheet) || ss.insertSheet(data.sheet); if (data.action === 'append') { sheet.appendRow(data.row); } else if (data.action === 'write') { sheet.clearContents(); data.rows.forEach(function(r){ sheet.appendRow(r); }); } return ContentService.createTextOutput(JSON.stringify({ok:true})).setMimeType(ContentService.MimeType.JSON); } function doGet(e) { var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheetByName(e.parameter.sheet); if (!sheet) return ContentService.createTextOutput(JSON.stringify({rows:[]})).setMimeType(ContentService.MimeType.JSON); var rows = sheet.getDataRange().getValues(); return ContentService.createTextOutput(JSON.stringify({rows:rows})).setMimeType(ContentService.MimeType.JSON); }

部署為網頁應用程式

點選「部署」→「新增部署」→「網頁應用程式」→「任何人均可存取」→「部署」，複製 URL 貼到上方欄位

📊 積分規則

名次積分 (A)：第1名 = 6分 / 第2名 = 3分 / 第3名 = 1分
MVP 積分公式：(A)×33% + (B名次積分)×33% + (C近期參賽率)×34%
最低出席場次： 場（適用於比率類獎項）
近期參賽率：取最後 6 場計算
適用：臭緊B、藏鏡人、和平大使、孔融讓梨、PK King 等
前三名可拍照
