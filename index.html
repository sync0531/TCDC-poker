☁️ 雲端同步（Supabase）

所有人只要開啟同一個網址就能自動共用資料，不需要任何伺服器。
每 30 秒自動同步，儲存時即時推送。

建立免費帳號

前往 supabase.com → Start for free → 用 GitHub 或 Google 登入

建立新專案

New Project → 填專案名稱「tcdc-poker」、設定密碼、選地區（Singapore 最近）→ Create project（等約 1 分鐘）

建立資料表

左側 → SQL Editor → New query → 貼上下方 SQL → Run

CREATE TABLE poker_data ( id TEXT PRIMARY KEY DEFAULT 'main', data JSONB NOT NULL, updated_at TIMESTAMPTZ DEFAULT now() ); INSERT INTO poker_data (id, data) VALUES ('main', '{}'); ALTER TABLE poker_data ENABLE ROW LEVEL SECURITY; CREATE POLICY "Public read" ON poker_data FOR SELECT USING (true); CREATE POLICY "Public write" ON poker_data FOR ALL USING (true);

取得金鑰

左側 → Project Settings → API → 複製 Project URL 和 anon public key 貼到上方欄位

儲存並測試

點「儲存」→「測試」確認連線成功 → 把 URL 和 anon key 分享給其他人貼入設定頁即可

📊 積分規則

名次積分 (A)：第1名 = 6分 / 第2名 = 3分 / 第3名 = 1分
MVP 積分公式：(A)×33% + (B名次積分)×33% + (C近期參賽率)×34%
最低出席場次： 場（適用於比率類獎項）
近期參賽率：取最後 6 場計算
適用：臭緊B、藏鏡人、和平大使、孔融讓梨、PK King 等
前三名可拍照
